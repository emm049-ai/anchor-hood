# AnchorHood Website

A modern, responsive website for AnchorHood - a Zimbabwean company supporting the built industry through innovative solutions.

## Features

- **Modern Design**: Built with Next.js 14 and Tailwind CSS
- **Responsive**: Fully responsive design that works on all devices
- **E-Commerce Ready**: Structure in place for future e-commerce integration
- **Booking System**: Appointment booking form for consultations
- **Service Showcase**: Comprehensive display of all services
- **Contact Forms**: Multiple ways to get in touch

## Pages

- **Home**: Hero section, services preview, values, and company overview
- **About**: Company vision, mission, values, team, and who we serve
- **Services**: Detailed breakdown of all services offered
- **Shop**: Product/service catalog (ready for e-commerce integration)
- **Contact**: Contact information and appointment booking form

## Getting Started

### Prerequisites

- Node.js 18+ installed
- npm or yarn package manager

### Installation

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser

### Building for Production

```bash
npm run build
npm start
```
